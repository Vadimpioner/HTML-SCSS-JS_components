1. _animations - навешиваем класс в html на те елементы которые хотим анимировать
2. в css элементы которые собираемся анимировать, манипулирем значением transform
3. _active - задаем дефолтное значение(то значение которое должен занимать элемент,когда закончится анимация)
4. _animations-no-hide - если не хотиим анимацию при скроле вверх, то к елементу в html добовляем этот класс
